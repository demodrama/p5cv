p5cv
====

Scketch de processing que integra diferentes librerías de visión artificial para hacer servidores de TUIO en Processing (1.5) al estilo de Community Core Vision
Librerías que usamos:
- Processing (versión 1.5)
http://processing.org/
- Diewald CV Kit (detección de blobs)
http://thomasdiewald.com/processing/libraries/diewald_CV_kit/
- Java CV Pro (detección de blobs)
http://www.mon-club-elec.fr/pmwiki_reference_lib_javacvPro/pmwiki.php
- CL Eye Multicam (aunque no es estable...) (driver de la ps2 eye camera)
http://codelaboratories.com/research/view/cl-eye-platform-java-sample
- SimpleOpenNI (kinect para processing)
https://code.google.com/p/simple-openni/
- TuioLib (Protocolo multitouch TUIO en Java)
http://www.tuio.org/?software
- GLGraphics (filtros OpenGL en Processing )
http://glgraphics.sourceforge.net/
- Jama (cálculo matricial)
http://math.nist.gov/javanumerics/jama/
